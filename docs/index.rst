==========
JayWormNET
==========

JayWormNET is a lightweight Worms Armageddon WormNET server, which basically
is a bundle of specialized HTTP and IRC servers. JayWormNET provided with such features as:

* Multi-channel support
* White and ban-lists
* Native W:A charset support
* Extended customization through scripting
